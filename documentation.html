Setup Instructions

Ensure Python and Flask are installed on your machine. Clone the repository from GitHub and install dependencies using pip install -r requirements.txt. Set environment variables for the IBM Watson API key and service URL.

Running Locally

To run Goscribby locally, navigate to the project directory in your terminal and execute the command 'flask run'. Ensure your local server is running by visiting 'localhost:5000' in your web browser.

Deploying to Heroku

Deploy Goscribby to Heroku by creating a new Heroku app in your Heroku dashboard. Connect your GitHub repository to Heroku and set up automatic deployments. Push your local changes to GitHub to trigger a deployment.

Using the Demo

To use the demo, navigate to the '/demo' page, upload an audio file in MP3 format, and submit it for transcription. The results will be displayed on the page.

Troubleshooting

If you encounter issues with the IBM Watson API, verify your credentials and network settings. For problems with Flask or deployment, consult the Flask and Heroku documentation for detailed troubleshooting guides.
